Student Performance Dashboard (Power BI)
Project Overview

This project presents an interactive Student Performance Dashboard created using Power BI to analyze academic performance based on marks, attendance, subject-wise results, and student demographics.

Objectives
Analyze relationship between attendance and marks
Identify subject-wise performance trends
Compare performance across different classes
Visualize gender distribution
Identify top-performing students
Dataset Features

The dataset includes:

Student Name
Class
Subject
Marks
Attendance
Gender
Teacher
Exam Date
Dashboard Visualizations

The dashboard contains:

KPI Cards (Average Marks, Average Attendance, Total Students)
Scatter Plot (Attendance vs Marks relationship)
Bar Chart (Average Marks by Subject)
Column Chart (Average Marks by Class)
Histogram (Marks Distribution)
Pie Chart (Gender Distribution)
Student Ranking Table
Interactive Filters (Class, Subject, Teacher, Gender)
Tools Used
Power BI Desktop
CSV Dataset
File Included
pride.pbix (Power BI dashboard file)
Insights

Students with higher attendance generally show better academic performance. Subject-wise analysis helps identify strong and weak academic areas across classes.
